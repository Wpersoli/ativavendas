# AtivaVendas — Modelo com hero refinado v6

Mantido do modelo:
- Conversas animadas de WhatsApp
- Modal/simulação
- Formulário de checkout
- Catálogo, bônus, indicação e scripts

Correções v6:
- Mais espaço nas bordas esquerda e direita
- Conteúdo do topo menor e mais elegante
- Mais espaço entre texto e imagem
- Visual do topo reduzido proporcionalmente
- Imagem/objetos do topo sem corte nas pontas
- Blurs, sombras e animações reduzidos para melhor desempenho
- Movimento do hero somente por reação do mouse
- Partículas antigas desativadas
- Inputs com autocomplete, limites básicos e pattern de telefone
- vercel.json com headers de segurança para deploy estático

Observação:
Este projeto é estático. Não existe back-end real dentro do ZIP. Para produção, validações críticas, anti-spam, rate limit, sanitização e envio seguro do formulário devem ser feitos no servidor/API.
